Description

Multiplex assays of variant effect (MAVEs) can test thousands of sequence variants in a single experiment. To facilitate the application of MAVE experimental results, researchers with the Alliance of Variant Effects Alliance (AVE) have developed MaveDB ( https://www.mavedb.org ), a public repository for large-scale measurements of sequence variant impact.  While these measurements have strong potential to accelerate sequence variant interpretations, the initial lack of reporting standards has hindered their use.  In particular, the practice of reporting MAVE results relative to assayed sequences rather than accessioned genomic, transcript, or protein coordinates has complicated their use.  In response, the Data Coordination and Dissemination Workstream of the AVE has developed new methods to map MAVE data to human reference sequences.  These tracks present hundreds of MAVE experiments mapped to the human reference genome.

Genomes

We provide MAVE data mappings on the GRCh38 genome.

Display Conventions and Configuration

Each MAVE score set is represented as a heat map, with rows representing alternative alleles and columns representing genomic positions.  Each cell in the heat map is colored according to the MAVE score, with the highest scores shown in yellow and the lowest scores shown in purple.  Further data on each experiment is available through MaveDB.

Methods

Variant mapping from human score sets was accomplished in three sequential steps.  First, each MaveDB target sequence was aligned to the GRCh38 human genome assembly using BLAT.  Second, we identified compatible transcript or protein reference sequences from RefSeq based on overlapping coordinates, leveraging the  biocommons SeqRepo python package, the Universal Transcript Archive (UTA) database and GenomicMedLab Common Operations on Lots of Sequences (cool-seq-tool) python package. For MAVE score sets which could be mapped directly to a contiguous chromosomal region, such as those which interrogate regulatory or other noncoding elements, this step was skipped.  Lastly, we extracted the genomic coordinates to assemble genome browser tracks, using the UCSC Genome Browser’s BigHeat utility as adapted within an open source python software package.

Data Availability

All data are freely available for download via the MaveDB FastAPI-driven API, specified at https://api.mavedb.org/docs
